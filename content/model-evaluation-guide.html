Model Evaluation Metrics Cheat Sheet

A reference guide on classification metrics (precision, recall, F1, AUC), regression metrics (MSE, RMSE, R²), and confusion matrices.

📈 Model Validation

Accuracy alone is misleading, especially for imbalanced datasets (e.g. churn predictions or fraud detection). Use specialized metrics to evaluate model errors.

1. Classification Evaluation Metrics

Understand the trade-offs of the confusion matrix columns:

  • Precision: Out of all positive predictions, how many were correct? Minimize false positives.
    Precision = TP / (TP + FP)
  • Recall (Sensitivity): Out of all actual positive cases, how many did the model find? Minimize false negatives.
    Recall = TP / (TP + FN)
  • F1-Score: The harmonic mean of precision and recall. Ideal for imbalanced classes.
    F1 = 2 * (Precision * Recall) / (Precision + Recall)
  • ROC AUC (Area Under Curve): Evaluates performance across all probability thresholds. Scale is 0.5 (random guessing) to 1.0 (perfect classifier).
from sklearn.metrics import classification_report, roc_auc_score

# Compute all classification performance statistics
print(classification_report(y_true, y_pred))
print(f"ROC AUC: {roc_auc_score(y_true, y_prob):.3f}")

2. Regression Evaluation Metrics

Quantify distance predictions error scales:

  • Mean Absolute Error (MAE): Average of absolute residuals. Easy to interpret.
  • Mean Squared Error (MSE): Average of squared residuals. Heavily penalizes large error residuals.
  • Root Mean Squared Error (RMSE): The square root of MSE. Translates errors back to the target variable's unit scale.
  • R² (Coefficient of Determination): The percentage of variance in the target variable explained by the features.
from sklearn.metrics import mean_squared_error, r2_score
import numpy as np

mse = mean_squared_error(y_true, y_pred)
rmse = np.sqrt(mse)
r2 = r2_score(y_true, y_pred)

print(f"RMSE: {rmse:.3f}, R2: {r2:.3f}")
