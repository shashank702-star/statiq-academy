Linear Algebra Foundations

Linear Algebra is the mathematical language of data. From image pixel representations to high-dimensional text embeddings and neural network weight matrices, data is structured and transformed using linear algebra operations.

1. Vectors: The Elements of Data

A Vector is an ordered list of numbers representing features of a data point. In 2D space, a vector represents a direction and magnitude from the origin. In data science, a vector can have thousands of dimensions (e.g., word embeddings).

Vector Dot Product

The dot product multiplies corresponding entries of two equal-length vectors and sums the results, producing a single scalar. It is key for computing similarity (cosine similarity) and matrix operations.

v · w = Σ (vi * wi) = v1w1 + v2w2 + ... + vnwn 
# Vector operations in NumPy
import numpy as np

# Feature vectors for two users
user_a = np.array([4.0, 5.0, 1.0]) # Rating for [Action, Comedy, Drama]
user_b = np.array([5.0, 4.0, 0.0])

# Dot Product calculation
dot_prod = np.dot(user_a, user_b)
print(f"Dot Product: {dot_prod}") # (4*5) + (5*4) + (1*0) = 40.0

# Cosine Similarity: dot(a, b) / (norm(a) * norm(b))
cos_sim = dot_prod / (np.linalg.norm(user_a) * np.linalg.norm(user_b))
print(f"Cosine Similarity: {cos_sim:.4f}")

2. Matrices: Datasets and Weights

A Matrix is a 2D grid of numbers. In machine learning, a matrix can represent a full dataset (rows = samples, columns = features) or the weights of a neural network layer.

Matrix Multiplication

Multiplying a matrix $A$ (size $m \times n$) by a matrix $B$ (size $n \times p$) yields a matrix $C$ ($m \times p$). Each element $C_{i,j}$ is the dot product of row $i$ of $A$ and column $j$ of $B$.

Ci,j = Σ Ai,k * Bk,j

Transpose, Identity, and Inverse

  • Transpose ($A^T$): Flips a matrix over its diagonal, swapping rows and columns.
  • Identity ($I$): A square matrix with 1s on the diagonal and 0s elsewhere. Multiplying any matrix by $I$ leaves it unchanged ($AI = A$).
  • Inverse ($A^{-1}$): A matrix that, when multiplied by the original matrix, yields the identity matrix ($A A^{-1} = I$). Only square matrices with non-zero determinants have inverses.
# Matrix transformations in Python
A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])

# Matrix Multiplication
C = np.dot(A, B) # Or: A @ B
print("C = A @ B:\n", C)

# Transpose
print("A Transposed:\n", A.T)

# Inverse of A
A_inv = np.linalg.inv(A)
print("Inverse of A:\n", A_inv)
print("A @ Inverse(A):\n", np.round(A @ A_inv)) # Identity Matrix

3. Eigenvalues and Eigenvectors

For a square matrix $A$, a non-zero vector $v$ is an eigenvector if multiplying it by $A$ only scales the vector by a scalar factor λ (the eigenvalue).

A * v = λ * v

In data science, eigenvectors represent the axes of maximum variance in datasets, which forms the mathematical foundation of Principal Component Analysis (PCA) for dimensionality reduction.

# Eigendecomposition
matrix = np.array([[4, 2], [1, 3]])
eigenvalues, eigenvectors = np.linalg.eig(matrix)

print("Eigenvalues:", eigenvalues)
print("Eigenvectors (columns):\n", eigenvectors)
