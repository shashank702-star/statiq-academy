Calculus & Optimization Basics

Calculus is the engine behind learning in neural networks and machine learning models. By computing derivatives, we measure the rate of change of errors relative to model parameters, allowing us to iteratively adjust weights to optimize performance.

1. Derivatives & The Slopes of Functions

The **Derivative** of a function $f(x)$, denoted as $f'(x)$ or $\frac{df}{dx}$, measures the instantaneous rate of change of the function output relative to changes in its input. Visually, the derivative is the slope of the line tangent to the function curve at a specific point.

Key Rules of Calculus

  • Power Rule: $\frac{d}{dx}(x^n) = n x^{n-1}$ — e.g., $\frac{d}{dx}(x^3) = 3x^2$
  • Constant Rule: $\frac{d}{dx}(C) = 0$ — e.g., $\frac{d}{dx}(5) = 0$
  • Sum Rule: $\frac{d}{dx}(f(x) + g(x)) = f'(x) + g'(x)$
  • Product Rule: $\frac{d}{dx}(f(x) \cdot g(x)) = f'(x)g(x) + f(x)g'(x)$

2. Chain Rule & Partial Derivatives

In machine learning, functions are often composite ($f(g(x))$) and have multiple variables (such as neural network weights). To solve these, we use multi-variable calculus.

The Chain Rule

The Chain Rule computes the derivative of composite functions. It is the core mathematical mechanic of backpropagation in deep neural networks, where we pass error gradients backward through multiple hidden layers.

dy/dx = (dy/du) * (du/dx) — where y = f(u) and u = g(x)

Partial Derivatives

When a function depends on multiple variables (e.g., $f(w, b)$ where $w$ represents weights and $b$ represents bias), a partial derivative measures the rate of change with respect to one variable while holding all other variables constant. We use the symbol ∂ (curly d) to write partial derivatives: $\frac{\partial f}{\partial w}$ and $\frac{\partial f}{\partial b}$.

Gradient ∇f = [ ∂f/∂w1, ∂f/∂w2, ..., ∂f/∂wn ] 
# Symbolic Derivatives in Python using SymPy
import sympy as sp

x, w, b = sp.symbols('x w b')
# Define loss function: L = (w*x + b - 10)**2
loss = (w*x + b - 10)**2

# Take partial derivative of loss with respect to weight 'w'
partial_w = sp.diff(loss, w)
print("Partial derivative w.r.t w:")
print(partial_w) # 2*x*(b + w*x - 10)

# Evaluate at values: x=2, w=3, b=1
gradient_val = partial_w.subs({x: 2, w: 3, b: 1})
print(f"Gradient value at w=3: {gradient_val}")

3. Gradient Descent Optimization

Gradient Descent is an optimization algorithm that shifts model weights in the opposite direction of the gradient to minimize the loss (error) function. By constantly moving in the direction of steepest descent, we converge toward the global minimum error.

wnew = wold - η * ∇L(w)
η (eta) = Learning Rate (step size scaling factor)

The Learning Rate (η)

  • Too Small: Converging takes too long, requiring heavy computational times.
  • Too Large: The optimization steps overshoot, oscillating and failing to converge to the minimum.
