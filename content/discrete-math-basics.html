Discrete Mathematics & Logic Basics

Discrete mathematics deals with countable, distinct structures. It is essential for understanding algorithms, conditional statements, database joins, probability spaces, and boolean logic operations in computer systems.

1. Set Theory & Logic Operations

A **Set** is a collection of distinct elements. In data analytics and SQL, table queries are based directly on set operations.

  • Union ($A \cup B$): Elements in set A OR set B OR both. In SQL, this is represented by UNION.
  • Intersection ($A \cap B$): Elements in set A AND set B. In SQL, this is equivalent to an INNER JOIN.
  • Difference ($A \setminus B$): Elements in set A but not in set B. In SQL, this is represented by EXCEPT or LEFT JOIN ... WHERE right.id IS NULL.

Boolean Algebra & Logic Gates

Logic gates perform basic logical operations that underpin conditional programming control flow (if/else statements):

AND (∧): Returns True if BOTH inputs are True.
OR (∨): Returns True if AT LEAST ONE input is True.
NOT (¬): Inverts the input (True becomes False).
XOR (⊕): Returns True if EXACTLY ONE input is True (A != B). 
# Set operations and Logic gates in Python
set_a = {"Python", "SQL", "Tableau"}
set_b = {"SQL", "Git", "R"}

# Set Math
union = set_a.union(set_b) # OR
intersection = set_a.intersection(set_b) # AND
difference = set_a.difference(set_b) # subtraction

print("Union:", union)
print("Intersection:", intersection) # {'SQL'}
print("Difference (A-B):", difference) # {'Python', 'Tableau'}

# Boolean checks
x = True
y = False
print("x AND y:", x and y)
print("x OR y:", x or y)
print("x XOR y:", x != y)

2. Combinatorics: Permutations & Combinations

Combinatorics is the study of counting arrangements. It is essential for calculating sample spaces in probability and assessing statistical probabilities.

Factorial ($n!$)

The product of all positive integers less than or equal to $n$. Represents the number of ways to arrange $n$ objects in a sequence.

n! = n * (n-1) * (n-2) * ... * 1

Permutations ($P$) vs. Combinations ($C$)

  • Permutations: Arrangements where **order matters** (e.g., choosing a President and Vice President from a pool).
    P(n, r) = n! / (n - r)!
  • Combinations: Selections where **order does not matter** (e.g., choosing a committee of 2 from a pool).
    C(n, r) = n! / (r! * (n - r)!)
# Permutations and Combinations in math module
import math

n = 5
r = 2

# Permutations P(5, 2): 5! / 3! = 20
perms = math.perm(n, r)
print(f"Permutations: {perms}")

# Combinations C(5, 2): 5! / (2! * 3!) = 10
combs = math.comb(n, r)
print(f"Combinations: {combs}")

3. Summation (Σ) and Product (Π) Notations

Mathematical equations in machine learning papers heavily rely on these short-hand iteration representations:

Summation (Σ - Sigma)

Indicates adding a sequence of numbers from a lower limit to an upper limit.

Σi=1n xi = x1 + x2 + ... + xn

Product (Π - Pi)

Indicates multiplying a sequence of numbers from a lower limit to an upper limit (e.g., computing joint likelihood probabilities).

Πi=1n xi = x1 * x2 * ... * xn
