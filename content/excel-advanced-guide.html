Advanced Excel Formulas & Analytics

A structured walkthrough of essential functions, lookup array formulas, Pivot Table tricks, and Power Query workflows required for modern business analysts.

📊 The Spreadsheet Analytical Engine

Before moving databases to SQL or Python, Excel remains the fastest tool for prototyping datasets. Master these core mechanics to clean and model data with efficiency.

1. The Evolution of Lookups (XLOOKUP)

Forget old VLOOKUP constraints (which only search left-to-right). Use XLOOKUP for flexible bi-directional lookups, vertical/horizontal matrices, and default error handling.

=XLOOKUP(lookup_value, lookup_array, return_array, [if_not_found], [match_mode])

# Example: Fetching Customer Name by ID
=XLOOKUP("CS-1012", A2:A100, B2:B100, "Not Found")

2. Index & Match (The Classic Dual Lookup)

If your version of Excel does not support XLOOKUP, the INDEX + MATCH combination is the robust, fast alternative for dynamic lookups.

=INDEX(return_column, MATCH(lookup_value, search_column, 0))

# Example: Find Monthly Charges for CS-1012
=INDEX(D2:D100, MATCH("CS-1012", A2:A100, 0))

3. Conditional Summing & Counting (SUMIFS / COUNTIFS)

Aggregate metrics conditionally based on multiple criteria columns.

=SUMIFS(sum_range, criteria_range1, criteria1, criteria_range2, criteria2)

# Sum charges for customers with tenure > 12 months in the "Enterprise" tier
=SUMIFS(E2:E100, C2:C100, ">12", F2:F100, "Enterprise")

4. Pivot Tables & Power Query

⚡ Power Query ETL

Instead of manually copy-pasting and cleaning columns, use **Power Query** to build repeatable data cleansing pipelines:

  • Get Data: Import directories of text files or databases.
  • Transform: Split columns, extract characters, fill null values, and change data types.
  • Repeat: Refreshing data automatically runs all cleaning steps without manual intervention.
