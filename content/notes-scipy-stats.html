SciPy Stats Module Foundations

A reference guide on using Python's SciPy library to represent statistical distributions, run hypothesis tests, and compute confidence intervals.

🔬 Scientific Computing in Python

SciPy builds on NumPy arrays to offer robust statistical functions. The sub-module scipy.stats contains tools for continuous and discrete random variables.

1. Working with Probability Distributions

SciPy allows you to represent probability density functions (PDF), cumulative distribution functions (CDF), and percent point functions (PPF).

from scipy import stats

# 1. Normal Distribution N(0, 1)
norm_dist = stats.norm(loc=0, scale=1)

# Probability Density of x = 1.96
print(f"PDF at 1.96: {norm_dist.pdf(1.96):.4f}")

# Cumulative Probability of x <= 1.96
print(f"CDF at 1.96: {norm_dist.cdf(1.96):.4f}") # Outputs ~0.975

# PPF (inverse of CDF) - Find value for 97.5% cumulative probability
print(f"97.5th Percentile: {norm_dist.ppf(0.975):.2f}") # Outputs ~1.96

2. Running Hypothesis Tests

Perform statistical significance calculations instantly using built-in tests:

One-Sample t-test

Compare a sample mean against a known population mean.

# Test if sample mean differs from population mean of 100
sample_data = [98, 102, 104, 99, 101, 97, 103, 105, 100, 101]
t_stat, p_val = stats.ttest_1samp(sample_data, popmean=100)

print(f"t-statistic: {t_stat:.4f}, p-value: {p_val:.4f}")

Two-Sample Independent t-test

Compare means of two distinct groups (common in A/B testing continuous metrics like session duration).

group_A = [12, 15, 14, 13, 16]
group_B = [18, 17, 19, 16, 20]

t_stat, p_val = stats.ttest_ind(group_A, group_B)
print(f"Ind. t-stat: {t_stat:.4f}, p-value: {p_val:.4f}")

3. Chi-Square Test of Independence

Used to determine if there is a significant association between two categorical variables.

# Contingency table (Device vs Churn)
# Columns: Churned, Retained. Rows: Mobile, Desktop
contingency_table = [[120, 380], 
                     [80, 420]]

chi2, p_val, dof, expected = stats.chi2_contingency(contingency_table)
print(f"Chi2 statistic: {chi2:.4f}, p-value: {p_val:.4f}")
