Mathematics & Calculus Foundations

Machine learning models represent data using linear algebra vectors, and optimize parameters using calculus derivatives. This guide covers core mathematical structures.

1. Vectors & Matrices (Linear Algebra)

A **Vector** is an ordered list of numerical features representing a single data point. A **Matrix** is a two-dimensional grid of numbers representing a full dataset table.

Vector v = [x1, x2, x3]
Matrix A = [ [a, b], [c, d] ]

Matrix multiplication combines feature inputs with weights to output predictions in deep neural networks.

# Vector Dot Product in Python (numpy)
import numpy as np

# Declaring vectors
v1 = np.array([2, 4, 6])
v2 = np.array([1, 3, 5])

# Calculate dot product: (2*1) + (4*3) + (6*5) = 2 + 12 + 30 = 44
dot_product = np.dot(v1, v2)
print(f"Dot Product: {dot_product}")  # Dot Product: 44

2. Derivatives & Rates of Change (Calculus)

A **Derivative** ($f'(x)$ or $\frac{dy}{dx}$) measures the rate at which a function changes relative to its input. In machine learning, derivatives represent slopes indicating the direction to move parameters to minimize error.

The Power Rule

For any variable raised to a constant exponent $n$:

d/dx( x^n ) = n * x^(n - 1)

Example: The derivative of $x^3$ is $3x^2$. The derivative of $5x^2 + 2x$ is $10x + 2$.

3. Gradient Descent Optimization

Gradient descent is an iterative optimization algorithm used to minimize loss functions (errors). It uses derivatives to find the steepest downward slope, adjusting weights step-by-step.

New Parameter = Current Parameter - Learning Rate * Derivative

If the derivative is positive, the parameter decreases. If negative, it increases, systematically shifting weights toward the minimum error boundary.
