Computer Vision & CNNs

Computer Vision enables machines to extract details from images and videos. This guide covers how images are represented as numerical grids and how Convolutional Neural Networks process them.

1. Images as Tensors

An image is represented as a three-dimensional grid (tensor) of shape: [Height, Width, Channels]. Grayscale images have 1 color channel, while color images have 3 channels: Red, Green, and Blue (RGB). Each cell value is an integer between 0 (black) and 255 (white).

# Representing RGB image grids in Python
import numpy as np

# Create a random RGB image grid (Height=32, Width=32, Channels=3)
image_tensor = np.random.randint(0, 256, size=(32, 32, 3))

# Check coordinates and channels
print(f"Shape: {image_tensor.shape}")  # Shape: (32, 32, 3)
print(f"Pixel Red channel at (0,0): {image_tensor[0, 0, 0]}")

2. Convolutional Layers (Kernel Filters)

Standard dense neural networks flatten images, losing spatial structures. **Convolutional layers** retain spatial structures by sliding small matrices (kernel filters, e.g., 3x3 grids) across images to detect features like edges, corners, and shapes.

The Convolution Operation

The filter multiplies overlapping pixel values element-wise, summing the values to yield a single value in the output **Feature Map**.

Feature Map Cell = Σ (Pixel_i,j * Kernel_i,j)

Different kernels detect different patterns: Sobel kernels detect vertical/horizontal edges, while Gaussian kernels blur images.

3. Pooling Layers (Downsampling)

Pooling layers reduce the dimensions of feature maps, lowering computational costs and making feature detection invariant to small shifts in position. The most common method is **Max Pooling**, which extracts the maximum value within a sliding grid window (e.g. 2x2 area).

Input Grid: [ [4, 9], [2, 7] ]
Max Pooling (2x2) Result: [ 9 ]

4. CNN Architecture

A typical image classification pipeline connects layers in a sequence:

  1. Convolutional + ReLU: Extract local image features.
  2. Max Pooling: Reduce size and keep strong features.
  3. Repeat: Stacking deeper layers to extract complex features (textures, objects).
  4. Flatten: Compress 2D feature maps into a 1D vector.
  5. Dense (Fully Connected): Classify the vector into target labels (e.g., Cat vs. Dog).
