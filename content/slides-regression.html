Introduction to Linear Regression

Linear Regression is one of the most fundamental algorithms in statistical machine learning. It models the linear relationship between a dependent variable (Y) and one or more independent variables (X).

Commonly used for forecasting trends, revenue projections, and uncovering mathematical correlations between features.

1 / 6

The Regression Line Equation

We represent the simple linear model with the equation:

Y = ββ + ββ * X + εε
  • ββ (Intercept): The value of Y when X is 0.
  • ββ (Slope): The rate of change of Y per unit change of X.
  • εε (Error/Residual): The random variation not captured by the line.

Ordinary Least Squares (OLS)

To find the best fitting line, we minimize the sum of squared differences (residuals) between predicted points and actual points.

Minimize RSS = Σ (y_i - ŷ_i)²

This is what the interactive regression plot on the StatIQ homepage lets you visualize! By adjusting the sliders, you manually solve OLS.

Evaluating Model Performance

We evaluate our predictions using metrics like:

  • R² (Coefficient of Determination): Represents the proportion of variance in Y explained by X (scale of 0 to 1).
  • MSE (Mean Squared Error): Average of the squared error terms. Bigger errors are penalized heavier.

Practical Python Code Guide

Implement Simple Linear Regression using Scikit-Learn:

# Fit linear model using Python
from sklearn.linear_model import LinearRegression
import numpy as np

X = np.array([[10], [15], [20], [30]]) # Study Hours
y = np.array([55, 68, 75, 89])        # Score

model = LinearRegression()
model.fit(X, y)

print(f"Slope (m): {model.coef_[0]:.2f}")
print(f"Intercept (c): {model.intercept_:.2f}")

Core Assumptions of OLS

For OLS estimates to be the Best Linear Unbiased Estimator (BLUE), the model must satisfy these assumptions:

  • Linearity: The relationship between X and Y must be linear.
  • Independence: Residuals must be independent of each other.
  • Homoscedasticity: Residuals must have constant variance at every level of X.
  • Normality: Residuals must be normally distributed.
