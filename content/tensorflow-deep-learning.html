TensorFlow & Deep Learning Architectures

A reference guide on building multi-layer neural networks, compiling models with optimizers, and running gradient descent fitting processes.

🧠 Deep Neural Networks

TensorFlow (with its high-level Keras API) allows you to define feedforward networks, convolutional neural networks (CNNs), and recurrent layers by stacking sequential classes.

1. The Sequential Model API

Learn how to define layers, activation functions (ReLU, Sigmoid), and regularization layers (Dropout) to prevent overfitting.

import tensorflow as tf
from tensorflow.keras import layers, models

# 1. Initialize Sequential Model architecture
model = models.Sequential([
    # Input layer with 10 features, feeding to 64 neurons
    layers.Dense(64, activation='relu', input_shape=(10,)),
    
    # Dropout layer to randomly disable 20% of weights during training
    layers.Dropout(0.2),
    
    # Second hidden layer
    layers.Dense(32, activation='relu'),
    
    # Output layer with 1 node for binary classification (probability)
    layers.Dense(1, activation='sigmoid')
])

model.summary()

2. Compiling the Network

The compilation step binds loss functions, gradient descent optimizers (Adam, SGD), and evaluation metrics to the model graph.

model.compile(
    # Adam optimizer with custom learning rate
    optimizer=tf.keras.optimizers.Adam(learning_rate=0.001),
    
    # Loss function for binary class classifications
    loss='binary_crossentropy',
    
    # Metrics to track during training epochs
    metrics=['accuracy']
)

3. Fitting & Evaluating

Fit the parameters (weights/biases) to the training datasets and run predictions on test sets.

# Fit model over 50 epochs (iterations through dataset)
history = model.fit(
    X_train, y_train, 
    epochs=50, 
    batch_size=32, 
    validation_split=0.2
)

# Evaluate performance on unseen test data
test_loss, test_acc = model.evaluate(X_test, y_test)
print(f"Test Accuracy: {test_acc * 100:.2f}%")
