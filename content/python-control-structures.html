Python Control Structures

Control structures dictate the order in which code statements are executed. Python uses comparison statements to branching logical decisions and loops to repeat instructions efficiently.

1. Conditionals: if, elif, and else

Conditional blocks evaluate expressions that yield Booleans (True or False). Python uses the if, elif (else if), and else keywords to define conditional code paths.

# Conditional statements
performance_rating = 4.2

if performance_rating >= 4.5:
    bonus_multiplier = 0.15
elif performance_rating >= 4.0:
    bonus_multiplier = 0.10
else:
    bonus_multiplier = 0.05

print(f"Bonus: {bonus_multiplier * 100}%") # Bonus: 10.0%

2. Comparison & Logical Operators

Multiple boolean expressions can be evaluated together using logical operators:

  • Comparison Operators: == (equals), != (not equal), >, <, >=, <=.
  • Logical Operators: and (both must be true), or (at least one must be true), not (inverts value).
# Combining conditions
revenue = 120000
profit_margin = 0.18

if revenue > 100000 and profit_margin >= 0.15:
    print("Company meets the High-Efficiency status.")
else:
    print("Company does not meet status.")

3. Loops: for, while, and range

Loops run a code block multiple times:

  • for loops: Iterate over a sequence (such as a list, string, or range of numbers).
  • while loops: Run as long as a specified condition remains True.
  • range(start, stop, step): Generates a sequence of integers (excludes stop value).
# For loop with range
for i in range(1, 4):
    print(f"Processing batch {i}...")

# While loop
countdown = 3
while countdown > 0:
    print(f"Ticks left: {countdown}")
    countdown -= 1

4. Loop Control: break, continue, and pass

  • break: Terminates the loop immediately.
  • continue: Skips the rest of the current iteration and moves to the next one.
  • pass: A null statement used as a placeholder where syntax requires code but no action is needed.
# Control keywords in loops
numbers = [1, 2, -99, 4, 5]

for n in numbers:
    if n == -99:
        print("Encountered corrupt value. Stopping loop.")
        break  # Exit loop
    if n % 2 == 0:
        continue  # Skip even numbers
    print(f"Odd positive number: {n}")
