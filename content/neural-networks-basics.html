Neural Networks Foundations

Neural networks form the backbone of deep learning. Inspired by biological brains, they stack layer networks to learn non-linear patterns from complex dataset features.

1. The Artificial Neuron (Perceptron)

A single neuron takes multiple numeric inputs ($x_i$), multiplies them by learned weights ($w_i$), adds a bias term ($b$), and passes the result through an activation function ($f$).

Neuron Output = f( Σ(x_i * w_i) + b )

Weights control the strength of input signals, while the bias allows shifting the activation function trigger point.

2. Activation Functions

Activation functions introduce non-linearities, allowing neural networks to model non-linear boundaries. Without them, stacking layers would behave as a single linear transformation.

Activation Equation Range Use Case
Sigmoid 1 / (1 + e^-x) (0, 1) Output layer of binary classifiers.
ReLU max(0, x) [0, ∞) Hidden layers (stops vanishing gradient).
Softmax e^x_i / Σ e^x_j (0, 1) Output layer of multi-class classifiers (probabilities sum to 1).

3. Stacking Layers: Multi-Layer Perceptrons

A standard Feedforward Neural Network (MLP) consists of three layer structures:

  • Input Layer: Receives the raw dataset features.
  • Hidden Layer(s): Extract abstract representations. More layers equal deeper representation limits.
  • Output Layer: Yields predictions (e.g. churn category, house price, dog/cat class labels).

4. Training via Backpropagation

Neural networks learn through a repeating loop of forward steps and feedback adjustments:

# Simulated Training Loop Concept in Python
# 1. Forward Pass: Calculate predictions
y_pred = model.predict(X_train)

# 2. Loss Computation: Measure error
loss = mean_squared_error(y_true, y_pred)

# 3. Backward Pass (Backpropagation):
# Calculate the gradient of the loss relative to each weight using the Chain Rule.
gradients = compute_gradients(loss)

# 4. Optimization: Adjust weights using Gradient Descent
weights -= learning_rate * gradients
