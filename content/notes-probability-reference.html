1. Bayes' Theorem

Bayes' Theorem updates the probability of a hypothesis (A) based on new evidence (B).

P(A|B) = [ P(B|A) * P(A) ] / P(B)
  • P(A|B): Posterior probability (probability of A given B).
  • P(B|A): Likelihood (probability of B given A).
  • P(A): Prior probability of A.
  • P(B): Marginal probability of evidence B.

📖 Guided Lesson: The Medical Test Paradox

Let's walk through a classic interview question using Bayes' Theorem: If a medical test for a disease has a 99% accuracy rate, and you test positive, what is the probability that you actually have the disease?

Step 1: Set the Prior Probabilities

Suppose the disease is rare, affecting only 0.1% of the population.
P(Disease) = 0.001
P(No Disease) = 0.999

Step 2: Define Test Accuracy (Likelihoods)

A 99% accuracy rate means:

  • 99% probability of testing positive if you have the disease: P(Positive | Disease) = 0.99
  • 1% false positive rate (testing positive when you don't have the disease): P(Positive | No Disease) = 0.01

Step 3: Calculate Marginal Probability P(Positive)

This is the sum of testing positive under all states:
P(Positive) = [P(Pos|Disease) * P(Disease)] + [P(Pos|No Disease) * P(No Disease)]
P(Positive) = [0.99 * 0.001] + [0.01 * 0.999] = 0.00099 + 0.00999 = 0.01098 (approx. 1.1%)

Step 4: Solve for P(Disease | Positive)

Apply Bayes' Theorem:
P(Disease | Positive) = [0.99 * 0.001] / 0.01098 = 0.00099 / 0.01098 = 0.0901

Result: Despite the test being 99% accurate, you only have a 9.01% chance of actually having the disease! This is because the disease is extremely rare, making false positives far more common than true positives.

2. A/B Testing Significance

A/B testing is statistical hypothesis testing applied to business decisions (e.g. testing conversion rates of two site layouts).

Two-Sample Z-Test for Proportions Formula

Z = (p_A - p_B) / sqrt( p * (1 - p) * (1/n_A + 1/n_B) )

Where p is the pooled conversion rate: p = (x_A + x_B) / (n_A + n_B)

📖 Guided Lesson: Page Layout A vs. B Conversion Test

Let's walk through determining if a new checkout button color significantly increased conversions.

Step 1: Set Hypotheses

- **Null Hypothesis ($H_0$)**: No difference in conversion rates ($p_A = p_B$).
- **Alternative Hypothesis ($H_1$)**: Layout B has a different conversion rate ($p_A ≠ p_B$).

Step 2: Collect Sample Data

  • Layout A (Control): $n_A = 10,000$ visitors, $x_A = 400$ conversions ($p_A = 4.0\%$)
  • Layout B (Variant): $n_B = 10,000$ visitors, $x_B = 460$ conversions ($p_B = 4.6\%$)

Step 3: Calculate the Pooled Rate (p) and Standard Error

p = (400 + 460) / (10,000 + 10,000) = 860 / 20,000 = 0.043 (4.3%)
Standard Error (SE) = sqrt( 0.043 * (1 - 0.043) * (1/10,000 + 1/10,000) ) = 0.00287

Step 4: Calculate the Z-Statistic and Compare to Alpha (5%)

Z = (0.040 - 0.046) / 0.00287 = -0.006 / 0.00287 = -2.09
Taking the absolute value, $|Z| = 2.09$.

Since $2.09$ is greater than the critical value $1.96$ (corresponding to a 95% confidence standard threshold $\alpha = 0.05$), we reject the null hypothesis. The 15% increase in conversion rate (from 4.0% to 4.6%) is statistically significant!
