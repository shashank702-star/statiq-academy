Data Analysis Foundations & EDA

A detailed reference for data classification systems, handling missing values, statistical outlier detection, and associations mapping.

1. Understanding Data Types

Before executing any statistical computations, you must classify your variables. Using incorrect metrics (e.g. calculating the average of zip codes) is a common analytical error.

Data Category Subtype Description Example
Qualitative (Categorical) Nominal Categories with no intrinsic order or ranking. Product Brand, Color, Country
Ordinal Categories with a clear, logical order. Customer Satisfaction (Poor, OK, Good)
Quantitative (Numerical) Discrete Countable whole values (no decimals possible). Website clicks, Support tickets, Orders count
Continuous Uncountable values within an interval (decimals). Store Revenue, Temperature, Wait time in sec

2. Cleansing Missing Data (Imputation)

Real-world datasets contain missing records. Cleanse them logically based on distribution shape:

  • Drop Missing Rows: Valid only if missingness is completely random (MCAR) and represents less than 5% of data.
  • Mean Imputation: Fill missing rows with column mean. Best for normally distributed, symmetric columns.
  • Median Imputation: Fill missing rows with column median. Best for highly skewed numeric data (e.g., Salaries, Transaction amounts) to avoid outlier bias.
  • Mode Imputation: Fill missing rows with the most common category. Best for categorical fields.

3. Mathematical Outlier Detection (Tukey's IQR)

Outliers distort statistical averages and predictive coefficients. Detect them systematically using Interquartile Range (IQR) fences.

The IQR Fences Equations

Identify the 25th percentile ($Q_1$) and 75th percentile ($Q_3$). Calculate $IQR = Q_3 - Q_1$.

Lower Boundary = Q1 - 1.5 * IQR
Upper Boundary = Q3 + 1.5 * IQR

Any values falling outside the interval [Lower Boundary, Upper Boundary] are flagged as outliers.

4. Measuring Correlation (Pearson's r)

Pearson's Correlation Coefficient ($r$) quantifies the strength and direction of the linear relationship between two variables, ranging from $-1.0$ (perfect negative correlation) to $+1.0$ (perfect positive correlation).

r = Cov(X, Y) / ( σ_X * σ_Y )

Remember: Correlation does not imply causation. Independent confounding factors (latent variables) must always be accounted for.
