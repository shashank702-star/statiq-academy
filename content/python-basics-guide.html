Python Programming Basics

Python is the default language for data engineering, analytics, and machine learning. This guide covers core programming structures to prepare you for writing analysis pipelines.

1. Variables & Primitive Data Types

Variables store values. Python is dynamically typed, meaning you don't need to declare types explicitly.

# Numeric Types
age = 25              # Integer (int)
salary = 85000.50     # Float

# Text and Boolean
name = "Alex Morgan"  # String (str)
is_enrolled = True    # Boolean (bool)

# Check variable types
print(type(age))      # <class 'int'>
print(type(salary))   # <class 'float'>

2. Control Flow: Conditionals & Loops

Use control structures to write conditional logic and repeat tasks.

# Conditionals
score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
else:
    grade = "C"

print(f"Grade: {grade}")  # Grade: B

# Loops: Repeating Operations
# range(5) returns numbers 0 to 4
for i in range(5):
    print(f"Iteration {i}")

# Iterating over lists
prices = [10, 20, 30]
total = 0
for price in prices:
    total += price
print(f"Total price: {total}")

3. Data Structures: Lists & Dictionaries

Lists and Dictionaries are the most common collection structures in Python for parsing dataset records.

# Lists: Ordered, mutable sequences of values
skills = ["Python", "SQL", "Tableau"]
skills.append("dbt")
print(skills[0])      # Python
print(skills[-1])     # dbt

# Dictionaries: Key-Value pairs (hash maps)
student = {
    "name": "Jane Doe",
    "track": "Data Science",
    "completed_lessons": 4
}
print(student["name"])  # Jane Doe
student["completed_lessons"] += 1
print(student)

4. Functions

Functions encapsulate code blocks, allowing reusability in data cleaning pipelines.

def calculate_tax(revenue, rate=0.15):
    """Calculates tax based on revenue and percentage rate."""
    tax = revenue * rate
    return tax

# Call the function
company_tax = calculate_tax(120000, 0.20)
print(f"Company Tax: ${company_tax}")  # Company Tax: $24000.0
