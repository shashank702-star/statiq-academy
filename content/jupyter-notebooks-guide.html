Jupyter Notebooks & Visual Explores

A reference guide on running Jupyter Notebooks, mastering keyboard shortcuts, integrating inline charts, and compiling markdown cells.

📓 Interactive Python Execution

Jupyter Notebooks combine live code blocks, mathematical equations, visualizations, and narrative text cells in a single JSON-based document structure.

1. The Two Main Modes

Jupyter cells behave differently depending on whether you are editing or selecting cells:

  • Command Mode (Blue border): Manipulate notebook structures. Press Esc to enter Command Mode.
  • Edit Mode (Green border): Write code or text inside cells. Press Enter to edit.

2. Essential Keyboard Shortcuts

Shortcut Action Mode
Shift + Enter Run current cell and advance to next. Any
Ctrl + Enter Run current cell in-place. Any
A Insert new cell above active cell. Command
B Insert new cell below active cell. Command
D + D (double press) Delete active cell. Command
M Convert cell type to Markdown (text/equations). Command
Y Convert cell type to Code (executable). Command

3. Magic Commands

Magic commands are special pre-compiled functions built into the IPython kernel to simplify system tasks.

# Render matplotlib charts inline below code cells
%matplotlib inline

# Measure execution speed of a code block
%%timeit
my_list = [x**2 for x in range(1000)]

# List files in the active workspace directory
%ls
